`capsule.html` — небольшое прозрачное окно только с капсулой (134×5px); клик по нему вызывает `goHome.`
`capsule-preload.js` — preload, который экспонирует goHome() для отправки IPC.
`main.js`:
> При открытии вики — создаётся/показывается окно капсулы как дочернее к основному, позиционируется внизу по центру.
>> При закрытии вики — окно капсулы скрывается.
>>> При resize — положение капсулы обновляется.
>>>> Обработчик capsule-go-home по IPC вызывает goHome() в основном окне.
>>>>> При hide/show основного окна капсула скрывается/показывается вместе с ним.
>>>>>> BrowserView занимает всю область контента (h = r.height, без вычета высоты полосы).
>>>>>>> `ipad-ui.js` — передаёт полные bounds без вычета высоты полосы; обработчик resize тоже использует полные bounds.
